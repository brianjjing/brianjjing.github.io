Alzheimer’s Diagnosis Model

The project that started it all - this was my first full end-to-end data science pipeline ever. As part of UC Irvine's 2023 Information and Computer Sciences (ICS) Summer Academy, I, along with a team of 6 others, developed a multinomial LASSO regression model and a random forest classification model to detect Alzheimer’s from medical records using R. We also used SMOTE for class balancing and ggplot2 for EDA beforehand. We ended up diagnosing a dataset of 2,700 Alzheimer's patients with 90% accuracy, presenting the model to ICS parents and UCI CS staff.
